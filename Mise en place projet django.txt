faire environement : 

python -m venv env
env/scripts/activate

install django 
pip install django


faire requirements.txt 

pip freeze > requirements.txt

code de base d un projet : 
django-admin startproject nomDuProjet .

lancer le serveur : 
python manage.py runserver

Mirgation et configuration de la bd : 

python manage.py makemigrations 
python manage.py migrate

installer application dans projets : 
python manage.py startapp nomApplication

mettre notre application dans le fichiers settings.py du projet : 

INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    'nomApplication',
]

Apres avoir fait un modele , on cree le fichier de migration :
python manage.py makemigrations

Puis on fait la migration:
python manage.py migrate

entrer dans le shell python : 
python manage.py shell

permet d executer du code dans le terminal , crtl + z pour quit

IMPORTANT : definir user avant d initialiser la base de donnee

creation superuser : 
python manage.py createsuperuser
